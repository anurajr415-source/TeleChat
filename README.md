# TeleChat Project Documentation

## Project Structure
- **backend/**: Contains the backend server code.
- **mobile/**: Contains the mobile application code.
- **admin/**: Contains the admin dashboard code.
- **monitoring/**: Contains code and setups for AI monitoring features.

## Installation Instructions
1. Clone the repository: `git clone https://github.com/anurajr415-source/TeleChat.git`
2. Navigate to each directory (`backend`, `mobile`, `admin`) and run the necessary installation commands (e.g., `npm install`, `pip install -r requirements.txt`).

## How to Run the Backend Server
1. Navigate to the `backend` directory.
2. Make sure all dependencies are installed.
3. Run the server: `npm start` or `python app.py` depending on the framework used.

## How to Run the Mobile App
1. Navigate to the `mobile` directory.
2. Make sure all dependencies are installed.
3. Execute the command to start the app: `npm start` or `react-native run-android` for Android.

## Admin Dashboard
1. Navigate to the `admin` directory.
2. Install necessary dependencies as instructed above.
3. Start the dashboard: `npm start`.

## AI Monitoring Features
- The `monitoring` directory includes scripts and configurations for AI monitoring functionalities.
- Ensure you set up the configs as per the instructions inside the directory.
